

A simple Finance Tracker app built using Kotlin for managing personal finances. This app allows users to track their income, expenses, and set budgets, helping them stay on top of their financial health.

Features

Track Income & Expenses: Add and categorize income and expenses.

Transaction History: View a list of all past transactions.

Monthly Budgeting: Set monthly budgets for each category and track spending.

Simple UI: User-friendly interface for easy finance management.

Technologies Used

Backend: Kotlin with Ktor (optional, if using backend)

Database: SQLite or Room for local data storage

UI: Android UI components (if using Android)
